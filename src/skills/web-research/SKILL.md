## Web Research Skill

When the user asks you to research a topic:

### Step 1: Plan Search Strategy
- Identify 3-5 key queries that cover different angles of the topic
- Prioritize authoritative sources (official docs, academic, established publications)

### Step 2: Gather Information
- Use `web_browse` to visit relevant URLs
- Take notes on key facts, statistics, and quotes from each source
- Track the source URL for each piece of information

### Step 3: Synthesize
- Cross-reference facts across sources
- Identify consensus vs. conflicting information
- Note any gaps in available information

### Step 4: Report
Format as a structured research brief:
1. **Executive Summary** (2-3 sentences)
2. **Key Findings** (bulleted, with source attribution)
3. **Analysis** (your synthesis of the information)
4. **Sources** (numbered list of URLs visited)

If the user specifies a format (memo, email, slides outline), adapt accordingly.
